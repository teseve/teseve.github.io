What is Tèsèvè?

Tèsèvè is a simple static webserver, powered by express, packaged in an app (using electron), for OSX and Windows.

Download and open the app, choose a path (or drop a folder on the app), and click on the link provided by the application : a simple static server is running from the given path.

Like the idea, but still want a command-line tool instead of a GUI? Take a look on sèvè!
